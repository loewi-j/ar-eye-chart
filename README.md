# eye-chart

## 项目介绍
本项目为【AR查视力】互动小游戏，应用的框架结构如下：
+ UI层，Vue
+ 互动层，createjs
+ 算法层，tf.js及knnClassifier模型与mobilenet模型

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
